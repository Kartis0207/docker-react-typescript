react の docker プロジェクト

参考：https://github.com/StephenGrider/docker-react

### docker

#### docker で当プロジェクトをビルド

```bash
docker-compose build
```

#### docker で当プロジェクトを起動

```bash
docker-compose up -d
```

これで接続できる
